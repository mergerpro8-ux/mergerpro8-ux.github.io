מיזוג PDF

Pro / חינמי מוגבל

חברו מספר קבצי PDF לקובץ אחד.

לחצו על "מיזוג PDF" במסך הראשי
בחרו 2 קבצים או יותר מהמכשיר
גררו לשינוי סדר (אופציונלי)
לחצו "מזג" — הקובץ המאוחד יישמר בתיקיית ההורדות
💡 ניתן לבחור עד 20 קבצים בפעולה אחת

סריקת מסמכים

Pro / חינמי מוגבל

סרקו מסמכים פיזיים ישירות ל-PDF באיכות מקצועית.

לחצו על "סריקה" במסך הראשי
כוונו את המצלמה למסמך — זיהוי גבולות אוטומטי יסמן את הדף
צלמו — האפליקציה תיישר, תחדד ותשפר את התמונה אוטומטית
הוסיפו עמודים נוספים או לחצו "צור PDF"
💡 לתוצאות מיטביות: סרקו במקום מואר עם רקע כהה מאחורי הדף
⚠️ הסורק דורש הרשאת מצלמה. ניתן לאשר בהפעלה הראשונה.

Merge PDFs

Pro / Limited Free

Combine multiple PDF files into a single document.

Tap "Merge PDF" on the home screen
Select 2 or more files from your device
Drag to reorder (optional)
Tap "Merge" — the combined file is saved to Downloads
💡 You can select up to 20 files in a single operation
📷

Document Scanner

Pro / Limited Free

Scan physical documents directly to professional-quality PDFs.

Tap "Scan" on the home screen
Point the camera at a document — auto edge detection will highlight the page
Capture — the app will automatically align, sharpen, and enhance the image
Add more pages or tap "Create PDF"
💡 For best results: scan in a well-lit area with a dark background behind the page
⚠️ The scanner requires camera permission, which can be granted on first launch.
✂️

Split PDF

Pro / Limited Free

Divide a PDF file into separate files.

Tap "Split" and select a PDF file
Choose the pages you want to extract
Tap "Split" — new files are saved to Downloads
✏️

PDF Editor

Pro / Limited Free

Add text, numbers, and elements on top of existing PDF documents.

Tap "Edit" and select a PDF file
Navigate to the desired page
Add text or numbers — drag to position
Tap "Save" to export the file
💡 Supports Hebrew, Arabic, and English — including bidirectional text (RTL)
✍️

Digital Signature

Pro / Limited Free

Sign documents directly from your device.

Create a signature by drawing or select a saved one
Select the PDF file
Position the signature on the page and adjust size
Save — the signature is embedded in the file
💡 Signatures are stored locally and are never sent to any server
🔒

Password Protection

Pro / Limited Free

Encrypt PDF files with a password (AES-256 encryption).

Select a PDF file
Set a password (minimum 4 characters)
Choose permissions: printing, copying, editing
Tap "Encrypt" — the protected file is saved
⚠️ Keep your password in a safe place — there is no way to recover it
🖼️

Images to PDF

Pro / Limited Free

Convert gallery images into a PDF file.

Tap "Images" and select one or more photos
Reorder if needed
Tap "Create PDF"
